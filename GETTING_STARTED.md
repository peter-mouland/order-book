# Getting Started

## Prerequisites:
- nvm (Node Version Manager)

## Initialise the Project

```bash
nvm use
pnpm i
```

## Running the Application Locally

```bash
pnpm dev
```

Open your browser and navigate to `http://localhost:5173` to view the application.

## Running Tests

```bash
pnpm test
```
